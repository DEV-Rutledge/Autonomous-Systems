# GPT Context

Read [`PROJECT_CONTEXT.md`](PROJECT_CONTEXT.md) first — this file only covers ChatGPT/Codex-specific notes on top of that.

## Role in This Venture

ChatGPT is used for early drafting and research passes (see how [`docs/BusinessPlan.md`](docs/BusinessPlan.md) and [`research/MarketResearch.md`](research/MarketResearch.md) came together) — Claude Code then reviews, fact-checks, reconciles against the rest of the repo, and implements. Don't treat a ChatGPT-drafted file as final; it goes through the same review-and-merge process as anything else before being trusted.

## When Drafting for This Venture

- Ground claims in real sources where possible (pricing, market size, benchmarks) — cite them, don't assert numbers as fact
- Flag assumptions that still need real validation rather than presenting them as settled
- Match the depth and structure of [`docs/BusinessPlan.md`](docs/BusinessPlan.md) if drafting a companion document — frontmatter, TOC, cross-links, real reasoning rather than bullet-point stubs
