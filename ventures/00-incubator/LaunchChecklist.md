# Launch Checklist

**Stage 13 of the [Incubator Pipeline](README.md#pipeline)** — used after an idea has been promoted to its own `ventures/NN-venture-slug/` folder and built (stage 12). This is the final gate before it goes live.

## Checklist

- [ ] Domain purchased
- [ ] Landing page
- [ ] Stripe configured
- [ ] Authentication
- [ ] Analytics
- [ ] Monitoring
- [ ] CI/CD
- [ ] Documentation
- [ ] Beta users
- [ ] Feedback loop
- [ ] First paying customer
- [ ] Weekly KPI review

Copy this checklist into the venture's own `tasks/` folder (e.g. `ventures/NN-venture-slug/tasks/LaunchChecklist.md`) once promoted — see [README.md § How to Use This Folder](README.md#how-to-use-this-folder).
