# Validation Checklist: Automation Reselling

Filled-in copy of [`IdeaValidationChecklist.md`](../../IdeaValidationChecklist.md) (incubator stage 2).

- [x] **Problem is painful** — businesses lose real hours to manual processes (front-desk intake, reporting, CRM syncs); documented value framing exists ("an automation saving 8 hrs/week" — see [MarketResearch.md](MarketResearch.md#business-model--pricing))
- [x] **Customers already spend money on solving it** — real market pricing exists today: $5–15K migration projects, $500+/mo retainers, $1,500–3,000/mo pipeline value to agency clients
- [x] **Recurring revenue is possible** — maintenance retainers are the standard model, not a bolt-on
- [x] **Automation potential is >80%** — the deliverable itself is automation; the delivery *process* (discovery → scope → build → handoff → retainer) is templatable per engagement even though each build is bespoke
- [x] **Can reach a first customer in <60 days** — no product-build phase; a portfolio piece takes 1–2 weeks, well within Ryan's 10–20 hrs/week availability
- [x] **API-first architecture is feasible** — Make.com, n8n, and Azure Functions are all webhook/API-native by design
- [x] **AI provides a durable advantage** — AI-agent-enhanced automations (not static workflows) are a real premium tier, and the Microsoft 365/Teams/Azure niche angle is a genuine differentiator most generic n8n/Make agencies can't offer
- [x] **Clear acquisition channel exists** — local business networks, LinkedIn outreach, agency partnerships, plus Ryan's existing professional network in the M365/Teams ecosystem
- [x] **Expansion opportunities are identifiable** — SMB → agency-reseller → enterprise M365 clients; chatbot/AI-agent work becomes a premium service line within this business rather than a separate venture

**Result:** clears the fast-fail gate cleanly on all 9 items — proceed to full scoring (see [Scorecard.md](Scorecard.md)).
